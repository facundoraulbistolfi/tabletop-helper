# Reglas oficiales de referencia — Chinchón (variante argentina)

Este archivo define las reglas que debe respetar **Chinchón Lab** en este repositorio.

## Mazo

- Se juega con **50 cartas**.
- Incluye **2 comodines**.

## Reparto inicial

- Se reparten **7 cartas por jugador**.
- El jugador que empieza recibe **8 cartas** y arranca descartando 1.

## Turno

- En cada turno, el jugador roba **1 carta** del mazo o del descarte.
- Luego descarta **1 carta**.
- El comodín **no puede descartarse nunca**.

## Juegos válidos

- **Escalera**: 3 o más cartas consecutivas del mismo palo.
- **Grupo**: 3 o más cartas del mismo número.
- Todo juego válido debe tener al menos **3 cartas**.

## Corte

- Se puede cortar solo si:
  - hay como máximo **1 carta sobrante**, y
  - el valor total del resto no supera **5 puntos**.
- No se puede cortar con resto mayor a 5.
- El comodín no puede ser la carta que se tira para cortar.

## Comodines

- El comodín nunca se puede tirar.
- No se puede usar ni para descartar ni para cortar.
- Un comodín que no forma parte de ningún juego cerrado vale **50 puntos** en contra.

## Chinchón

- El chinchón solo vale como chinchón si está hecho **sin comodines**.
- Si las 7 cartas forman una corrida usando comodín, no cuenta como chinchón:
  vale **-10 puntos**, pero no gana automáticamente.

## Puntaje y eliminación

- La partida es a **100 puntos**.
- Un jugador queda eliminado al llegar a **100 puntos o más**.
